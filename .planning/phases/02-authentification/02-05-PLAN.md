  <action>
1. Add an OTP model to `prisma/schema.prisma` (in the USERS & AUTH section):
   ```prisma
   model PhoneOtp {
     id        String   @id @default(cuid())
     userId    String
     phone     String
     code      String
     expiresAt DateTime
     usedAt    DateTime?
     attempts  Int      @default(0)
     createdAt DateTime @default(now())

     @@index([userId])
     @@index([phone, code])
     @@map("phone_otps")
   }
   ```
   Run `npx prisma generate`. Note db:push as needed.

2. Create `src/lib/sms.ts`:
   - Define interface:
     ```ts
     export interface ISmsService {
       sendOtp(phone: string, code: string): Promise<boolean>;
     }
     ```
   - Create SimulatedSmsService class implementing ISmsService:
     * sendOtp: logs `[SMS OTP] Code ${code} sent to ${phone}` to console, returns true
     * In dev mode, this is the active implementation
   - Export singleton: `export const smsService: ISmsService = new SimulatedSmsService()`
   - Add comment: "// Replace with TwilioSmsService or VonageSmsService for production"

3. Create `src/features/auth/actions/send-otp.ts`:
   - "use server"
   - Accept userId and phone
   - Generate 6-digit code: `Math.floor(100000 + Math.random() * 900000).toString()`
   - Invalidate any previous OTPs for this user: update all existing PhoneOtp where userId and usedAt is null to usedAt = now
   - Create PhoneOtp record: { userId, phone, code, expiresAt: new Date(Date.now() + 5 * 60 * 1000) } -- 5-minute expiry
   - Call smsService.sendOtp(phone, code)
   - Return ActionResult success

4. Create `src/features/auth/actions/verify-otp.ts`:
   - "use server"
   - Accept userId and code
   - Find the most recent unused PhoneOtp for this userId where expiresAt > now
   - If not found: return error "Code invalide ou expire"
   - If attempts >= 5: return error "Trop de tentatives. Demandez un nouveau code."
   - If code doesn't match: increment attempts on the OTP record, return error "Code incorrect"
   - If code matches: mark OTP as used (usedAt = now), update user phoneVerified = true
   - Return ActionResult success
  </action>
