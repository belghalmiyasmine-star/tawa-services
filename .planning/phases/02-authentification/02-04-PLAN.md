1. Create `src/features/auth/actions/forgot-password.ts`:
   - "use server"
   - Accept email from forgotPasswordSchema validation
   - Find user by email -- if not found, still return success (don't reveal if email exists for security)
   - If user found: generate token (crypto.randomUUID()), create PasswordReset record with expiresAt = now + 1 hour
   - Invalidate any previous password reset tokens for this user (update all existing to usedAt = now)
   - Call sendPasswordResetEmail(email, token, locale)
   - Return ActionResult success with message "Si cet email est enregistre, vous recevrez un lien de reinitialisation"

2. Create `src/features/auth/actions/reset-password.ts`:
   - "use server"
   - Accept token + newPassword + confirmPassword from resetPasswordSchema validation
   - Find PasswordReset by token: must exist, not expired, not used
   - Hash new password with bcryptjs.hash(password, 12)
   - Update user: prisma.user.update({ where: { id: passwordReset.userId }, data: { passwordHash } })
   - Mark token as used: prisma.passwordReset.update({ where: { id }, data: { usedAt: new Date() } })
   - Return ActionResult success

3. Create `src/features/auth/components/ForgotPasswordForm.tsx`:
   - "use client"
   - react-hook-form with forgotPasswordSchema
   - Email input field
   - Submit button "Envoyer le lien"
   - On success: show confirmation message (same message whether email exists or not)
   - Link back to login page

4. Create `src/features/auth/components/ResetPasswordForm.tsx`:
   - "use client"
   - react-hook-form with resetPasswordSchema (password + confirmPassword)
   - Token read from URL searchParams and passed as hidden field
   - Password fields with show/hide toggle
   - Password strength indicator (reuse pattern from PasswordStep)
   - Submit button "Reinitialiser mon mot de passe"
   - On success: show "Mot de passe reinitialise avec succes" and link to login
   - On error: show specific error (token expired, already used)

5. Create `src/app/[locale]/(client)/auth/forgot-password/page.tsx`:

   - If no token in URL, show error message
