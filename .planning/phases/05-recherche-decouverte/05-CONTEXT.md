# Phase 5: Recherche & Decouverte — Context

## User Design Decisions

### Homepage Search
- Search bar with autocomplete suggestions (categories + services)

### Search Results Page
- Left sidebar filters: category, city, price range min/max, rating, price type, verified toggle
- Collapsible as Sheet on mobile
- Results displayed as grid of ServiceCards
- Sort options: relevance, price asc/desc, rating, newest
- Pagination at bottom

### Category Browsing Page
- Grid of 10 category cards with emoji icons and service count
- Click to see subcategories as filter chips + services grid

### Service Detail Page
- Image gallery
- Title, category badge, price in TND
- Provider mini-card (photo, name, rating, city, verified badge)
- Full description, inclusions/exclusions
- "Reserver" button for fixed/hourly pricing
- "Demander un devis" for quote type
- "Contacter" button
- Similar services section at bottom

### Style & Layout
- All mobile-first responsive
- Same Airbnb rounded style used throughout the project
