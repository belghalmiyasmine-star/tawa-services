1. Create src/features/admin/schemas/category-schemas.ts:
- createCategorySchema: z.object({ name: z.string().min(2).max(100), slug: z.string().min(2).max(100).regex(/^[a-z0-9-]+$/), icon: z.string().optional(), description: z.string().max(500).optional(), parentId: z.string().optional().nullable(), isActive: z.boolean().default(true), sortOrder: z.coerce.number().int().min(0).default(0) })
- updateCategorySchema: createCategorySchema.extend({ id: z.string() })

2. Create src/features/admin/actions/category-actions.ts ("use server"):
- All actions check ADMIN role.
- getCategoriesTreeAction(): fetch all categories (isDeleted=false) with children relation, ordered by sortOrder. Include _count of services for each. Return as flat array — tree structure built client-side.
- createCategoryAction(data): validate with createCategorySchema, check slug uniqueness, create category. Return ActionResult.
- updateCategoryAction(data): validate with updateCategorySchema, check slug uniqueness (excluding self), update. Return ActionResult.
- deleteCategoryAction(categoryId): soft-delete category (isDeleted=true, deletedAt=now). If has children, also soft-delete children. If category has services, return error "Cette categorie contient des services actifs".
- toggleCategoryActiveAction(categoryId): flip isActive boolean.

3. Create src/features/admin/components/CategoryDialog.tsx ("use client"):
