1. Create src/features/admin/schemas/content-schemas.ts:
- createFaqSchema: z.object({ question: z.string().min(10).max(500), answer: z.string().min(10).max(5000), category: z.enum(["general", "booking", "payment", "provider"]).default("general"), sortOrder: z.coerce.number().int().min(0).default(0), isActive: z.boolean().default(true) })

- createBannerSchema: z.object({ title: z.string().min(3).max(200), subtitle: z.string().max(500).optional(), imageUrl: z.string().url().optional(), linkUrl: z.string().url().optional(), position: z.enum(["homepage", "search", "category"]).default("homepage"), isActive: z.boolean().default(true), sortOrder: z.coerce.number().int().min(0).default(0), startDate: z.string().datetime().optional().nullable(), endDate: z.string().datetime().optional().nullable() })
- updateBannerSchema: createBannerSchema.extend({ id: z.string() })

2. Create src/features/admin/actions/content-actions.ts ("use server"):
- All actions check ADMIN role, return ActionResult<T>.
- getFaqsAction(): all FAQs (isDeleted=false) ordered by sortOrder, category.
- createFaqAction(data): validate, create FAQ. Return created FAQ.
- updateFaqAction(data): validate, update FAQ. Return updated FAQ.
- deleteFaqAction(id): soft-delete (isDeleted=true, deletedAt=now).
- getLegalPagesAction(): all legal pages. If empty, seed 3 default pages:
  - { slug: "cgu", title: "Conditions Generales d'Utilisation", content: "Contenu a rediger..." }
  - { slug: "privacy", title: "Politique de Confidentialite", content: "Contenu a rediger..." }
  - { slug: "legal-mentions", title: "Mentions Legales", content: "Contenu a rediger..." }
  Use upsert by slug to be idempotent.
- updateLegalPageAction(data): validate, update legal page content + title + updatedBy.
- getBannersAction(): all banners (isDeleted=false) ordered by sortOrder.
- createBannerAction(data): validate, create banner.
- updateBannerAction(data): validate, update banner.
- deleteBannerAction(id): soft-delete.
- toggleBannerActiveAction(id): flip isActive boolean.

3. Create src/features/admin/components/FaqEditor.tsx ("use client"):
- Accept faqs[] prop and parentCategories for filtering.
- Tab bar to filter by category: "Tous", "General", "Reservation", "Paiement", "Prestataire".
- List of FAQ cards, each showing:
  - Question (bold), Answer (truncated), Category badge, Active switch
  - Edit button -> opens Dialog with form (question textarea, answer textarea, category select, sortOrder input, active switch)
  - Delete button -> AlertDialog confirmation -> deleteFaqAction
- "Ajouter une FAQ" button -> opens empty Dialog for creation.
- Form uses react-hook-form + zodResolver(createFaqSchema or updateFaqSchema).
- On save: call createFaqAction or updateFaqAction, toast success, refresh.
- Use useTranslations("admin.content").

- On save: call updateLegalPageAction, toast success.
- Form per page, independent saves (no global form).
- Use useTranslations("admin.content").
- Show "Derniere modification: {date}" below each save button.
