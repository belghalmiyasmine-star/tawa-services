1. Create `src/features/payment/actions/invoice-actions.ts`:
   - "use server" directive

   - Define `InvoiceData` interface:
     ```
     {
       invoiceNumber: string;      // TAWA-INV-{YYYYMMDD}-{short-id}
       date: string;               // ISO date of releasedAt
       clientName: string;
       clientEmail: string;
       providerName: string;
       providerDisplayName: string;
       serviceTitle: string;
       serviceDescription: string;
       amount: number;             // Total amount
       commission: number;         // 12% commission
       netAmount: number;          // providerEarning
       paymentMethod: string;
       referenceNumber: string;    // payment.id
       bookingId: string;
       scheduledAt: string;
     }
     ```

   - `getInvoiceDataAction(bookingId: string): Promise<ActionResult<InvoiceData>>`:
     - Validate session (allow CLIENT who owns booking OR PROVIDER who owns booking via provider.userId)
     - Fetch payment with booking, booking.service, booking.client (User), booking.provider with user relation
     - If payment.status not in [HELD, RELEASED]: return error "Facture non disponible"
     - Generate invoiceNumber: `TAWA-INV-${format(releasedAt||createdAt, 'yyyyMMdd')}-${payment.id.slice(-6).toUpperCase()}`
     - Map to InvoiceData and return

   - `getMonthlyStatementAction(month: string): Promise<ActionResult<MonthlyStatementData>>`:
     - month param format: "YYYY-MM"
     - Validate PROVIDER session
     - Parse month into start/end date range
     - Query all RELEASED payments for provider in that month
     - Aggregate: totalMissions, grossRevenue, totalCommission, netEarnings
     - List individual transactions
     - Return { period: month, provider info, summary, transactions[] }

   - Props: `InvoiceData & { viewAs: "client" | "provider" }`
   - Professional invoice layout:
     - Header: "TAWA SERVICES" logo/text, "FACTURE" title
     - Invoice number and date (top right)
     - Two columns: "Facture a" (client info) and "Prestataire" (provider info)
     - Table: Service description, quantity (1), unit price, total
     - If viewAs=provider: show commission deduction row and net amount
     - If viewAs=client: show total only
     - Footer: "Mode demonstration — document de simulation"
     - Print button: `window.print()` — hidden in @media print
   - Add CSS for @media print: hide non-essential elements (nav, footer, print button), white bg, clean margins
