1. Create `src/features/payment/services/payment-service.interface.ts`:
   - Define `ProcessPaymentInput` type: `{ bookingId: string; method: PaymentMethod; amount: number }`
   - Define `ReleasePaymentInput` type: `{ bookingId: string }`
   - Define `RefundPaymentInput` type: `{ bookingId: string; amount: number }`
   - Define `PaymentResult` type: `{ success: boolean; referenceNumber: string; error?: string }`
   - Define `IPaymentService` interface with methods:
     - `processPayment(input: ProcessPaymentInput): Promise<PaymentResult>` — initiates payment (moves to HELD for escrow)
     - `releasePayment(input: ReleasePaymentInput): Promise<PaymentResult>` — releases held funds to provider minus 12% commission
     - `refundPayment(input: RefundPaymentInput): Promise<PaymentResult>` — processes refund
   - Import PaymentMethod from `@/types`

2. Create `src/features/payment/services/simulated-payment.service.ts`:
   - Class `SimulatedPaymentService implements IPaymentService`
   - `processPayment`:
     - Fetch payment by bookingId (payment has unique bookingId)
     - Validate payment exists and status is PENDING
     - Generate reference number: `TAWA-${Date.now().toString(36).toUpperCase()}-${Math.random().toString(36).slice(2, 6).toUpperCase()}`
     - Update payment: status=HELD, heldAt=now, method=input.method
     - Return { success: true, referenceNumber }
     - Log `[SimulatedPaymentService] processPayment for booking ${input.bookingId} — reference ${ref}`
   - `releasePayment`:
     - Fetch payment by bookingId where status=HELD
     - Calculate commission = amount * 0.12, providerEarning = amount - commission
     - Update payment: status=RELEASED, releasedAt=now, commission, providerEarning
     - Return { success: true, referenceNumber: payment.id }
   - `refundPayment`:
     - Fetch payment by bookingId
     - Update: status=REFUNDED, refundAmount=input.amount, refundedAt=now
     - Return { success: true, referenceNumber: payment.id }
   - Export singleton: `export const paymentService: IPaymentService = new SimulatedPaymentService()`
   - Import prisma from `@/lib/prisma`

3. Create `src/lib/validations/payment.ts`:
   - `processPaymentSchema`: z.object({ bookingId: z.string().cuid(), paymentMethod: z.enum(["CARD", "D17", "FLOUCI", "CASH"]) })
   - `checkoutFormSchema`: z.object for the card form: cardNumber z.string().regex(/^\d{16}$/), expiryDate z.string().regex(/^\d{2}\/\d{2}$/), cvv z.string().regex(/^\d{3}$/) — format validation only, no real processing
   - Export types with z.infer
