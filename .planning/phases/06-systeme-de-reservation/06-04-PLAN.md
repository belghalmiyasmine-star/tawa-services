**QuoteResponseCard (`src/features/booking/components/QuoteResponseCard.tsx`):**

"use client" component. Props: `quote` (id, status, description, proposedPrice, proposedDelay, expiresAt, respondedAt, service: { title }, createdAt).

Renders a card showing the quote status and details:
- **PENDING state:** "En attente de reponse du prestataire" with countdown timer showing time remaining until expiresAt (e.g., "Expire dans 36h 24min"). Yellow/amber border.
- **RESPONDED state:** Show provider's proposed price (formatted as "XX.XX TND"), proposed delay, and two buttons: "Accepter le devis" and "Decliner". Green border.
- **ACCEPTED state:** "Devis accepte" badge, show price, link to the created booking.
- **DECLINED state:** "Devis decline" badge. Gray border.
- **EXPIRED state:** "Devis expire — le prestataire n'a pas repondu dans les 48h". Red border.

Countdown timer: use a useEffect with setInterval (every 60s) to update remaining time display. No external library.

"Accepter" button opens QuoteAcceptFlow dialog/sheet.
"Decliner" button calls declineQuoteAction with confirmation dialog (AlertDialog).
