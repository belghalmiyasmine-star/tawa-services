1. Create `src/features/notification/actions/notification-queries.ts`:
   - Add "use server" directive.
   - `getNotificationsAction(params?: { filter?: "all" | "unread"; cursor?: string; limit?: number }): Promise<ActionResult<{ notifications: NotificationItem[]; nextCursor: string | null }>>`:
     - Get session, validate with getNotificationsSchema.
     - Build where clause: { userId: session.user.id, isDeleted: false }. If filter === "unread", add { read: false }.
     - Fetch with cursor-based pagination (orderBy: createdAt desc, take: limit + 1).
     - Return NotificationItem[] with: id, type, title, body, read, readAt, data (parsed JSON), createdAt.
   - `getUnreadNotificationCountAction(): Promise<ActionResult<{ count: number }>>`:
     - Count notifications where userId = session.user.id, read = false, isDeleted = false.
   - `getNotificationPreferencesAction(): Promise<ActionResult<NotificationPrefs>>`:
     - Fetch or create default NotificationPreference for current user.
     - Return { emailEnabled, inAppEnabled, quietHoursStart, quietHoursEnd, disabledTypes }.
   - Define TypeScript interfaces: `NotificationItem { id, type, title, body, read, readAt, data: Record<string, string> | null, createdAt }`, `NotificationPrefs { emailEnabled, inAppEnabled, quietHoursStart, quietHoursEnd, disabledTypes: string[] }`.

2. Create `src/features/notification/actions/notification-actions.ts`:
   - Add "use server" directive.
   - `markNotificationReadAction(notificationId: string): Promise<ActionResult<void>>`:
     - Get session, validate notificationId.
     - Update notification: set read = true, readAt = new Date(). Verify userId matches session.
   - `markAllNotificationsReadAction(): Promise<ActionResult<{ count: number }>>`:
     - Get session.
     - `prisma.notification.updateMany({ where: { userId: session.user.id, read: false, isDeleted: false }, data: { read: true, readAt: new Date() } })`.
     - Return count.
   - `updateNotificationPreferencesAction(prefs: { emailEnabled: boolean; inAppEnabled: boolean; quietHoursStart: string | null; quietHoursEnd: string | null; disabledTypes: string[] }): Promise<ActionResult<void>>`:
     - Get session, validate with updatePreferencesSchema.
     - Upsert NotificationPreference for current user.
     - Return success.
