1. Create `src/features/messaging/components/MessageBubble.tsx` ("use client"):
   - Props: `{ message: MessageItem; isFromMe: boolean; showReadReceipt: boolean }`.
   - Sent messages (isFromMe=true): aligned right, bg-blue-500 text-white, rounded-2xl rounded-br-sm.
   - Received messages: aligned left, bg-muted (gray in light mode), rounded-2xl rounded-bl-sm.
   - Below bubble: timestamp in text-xs text-muted-foreground using `Intl.DateTimeFormat('fr-TN', { hour: '2-digit', minute: '2-digit' })`.
   - If isFromMe and showReadReceipt: show small "Lu" text or double-check icon below timestamp when message.isRead is true, "Envoye" when false.
   - If message.flagged: show muted italic text "Message signale" instead of content.

2. Create `src/features/messaging/components/MessageInput.tsx` ("use client"):
   - Props: `{ conversationId: string; onMessageSent: () => void }`.
   - State: inputValue (string), isSending (boolean), error (string | null).
   - Textarea (auto-resize, max 3 lines) + Send button (lucide Send icon).
   - Submit handler: call sendMessageAction({ conversationId, content: inputValue }). On success, clear input, call onMessageSent(). On error with "contact_info_blocked", show toast with tMessaging("contactInfoBlocked") message using useToast. On other error, show tMessaging("errors.sendFailed") toast.
   - Send on Enter (without Shift). Shift+Enter for new line.
   - Disable send button when inputValue.trim() is empty or isSending.

3. Create `src/features/messaging/components/ChatView.tsx` ("use client"):
   - Props: `{ conversationId: string; currentUserId: string; otherUserName: string }`.
   - State: messages (MessageItem[]), isLoading (boolean), hasMore (boolean).
   - On mount: fetch initial messages via getConversationMessagesAction({ conversationId, limit: 30 }). Set messages. Call markMessagesAsReadAction(conversationId).
   - Polling: useEffect with setInterval every 5000ms. On each tick, fetch latest messages (no cursor = newest first). Compare with current state — if new messages found (by id), append and auto-scroll. If new messages from other user, call markMessagesAsReadAction.
   - Auto-scroll: useRef on messages container. scrollTo bottom on mount and when new messages arrive. Use scrollHeight comparison — only auto-scroll if user was already near bottom (within 100px).
   - Load older messages: "Charger les messages precedents" button at top. Uses cursor of oldest message to fetch previous page.
   - Render: Messages grouped by date (Aujourd'hui, Hier, or formatted date). Each message rendered via MessageBubble. showReadReceipt only on the LAST sent message (not all).
   - Handle onMessageSent callback from MessageInput: immediately add optimistic message to state, then refresh on next poll.
   - Clean up interval on unmount.

4. Create `src/features/messaging/components/ConversationList.tsx` ("use client"):
   - Props: `{ conversations: ConversationListItem[]; currentUserId: string }`.
   - Renders a list of conversation cards. Each card shows:
     - Avatar with first letter of otherUser.name (Avatar + AvatarFallback from shadcn).
     - otherUser.name in font-semibold.
     - Last message preview (truncated to 50 chars). If lastMessage.isFromMe, prefix with "Vous: ".
     - Relative timestamp of lastMessage.createdAt (using timeAgo helper: "A l'instant", "Il y a 5 min", "Il y a 2h", "Il y a 3j").
     - Unread count badge (if > 0): small bg-blue-500 text-white rounded-full with count.
     - Booking service title in text-xs text-muted-foreground below.
   - Each card is a Link to /messages/[conversationId] (client) or /provider/messages/[conversationId] (detect from URL).
   - Sorted by lastMessage.createdAt DESC (already sorted from server).
   - Empty state: illustration or icon + tMessaging("noConversations").
   - Polling: useEffect with setInterval every 5000ms to refresh conversation list via getConversationsAction. Update state if data changed (compare lastMessage.id or unreadCount).
